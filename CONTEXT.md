# Project Context: Eventos Ocaña

## Purpose
A platform to discover, manage, and promote events in the city of Ocaña. The goal is to provide a central hub for cultural, religious, commercial, and institutional events.

## Domain Language

- **Event (Evento)**: A scheduled occurrence with a date, location, category, and description.
- **Company (Empresa/Organizador)**: The entity responsible for creating and managing events.
- **Category (Categoría)**: A top-level classification (e.g., Religioso, Comercial, Institucional).
- **Sub-category (Sub-categoría)**: A more specific classification (e.g., Iglesia Católica, Iglesia Evangélica under Religioso).
- **Favorite (Favorito)**: A personalized bookmark for a user to track events they are interested in.
- **CEDIT**: The institutional branding and technical platform provider.

## Technical Rules

- Use **Supabase** for database and authentication.
- Styling must be **Vanilla CSS** or **Tailwind CSS** (as requested).
- Maintain a **Premium** and **Aesthetic** UI for all components.
- All images should be stored in Supabase Storage buckets.
